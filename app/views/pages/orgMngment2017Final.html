3. プロフェッショナルとは何か、またプロフェッショナル集団をマネジメントする方法を図示する

  1. プロフェッショナルとは
    1. 仕事人（✕：組織人）
    2. 組織に紐付かず、仕事を通じた自己実現・高次欲求充足を目指す
  2. プロフェッショナル集団をマネジメントする方法
    1. 組織　⇆　個人　⇆　仕事
    2. 組織　←（必要な範囲での貢献／高次欲求充足の条件獲得＋低次欲求の充足）→　個人
    3. 個人　←（高次欲求の充足／最大限の貢献）→　仕事
    4. 大学　←（授業）→　個人　←（研究）→　教授職

4. キャリアアンカーとは何か、また代表的なキャリアアンカーを複数取り上げ表にまとめ対比する

  1. キャリア・アンカーとは
    1. キャリアにおける選択、職業や職種、勤務先などを選ぶ際の「判断基準」
    2. 社会人のキャリア選択の「基本指針」であり、キャリアを「方向付ける役割」を果たす
    3. 自分らしく生きるために、自分のキャリアにおいて絶対に譲りたくない「最も大切なこと」
  2. 代表的なキャリアアンカーを複数
    1. 純粋な挑戦
      1. 挑戦、克服、競争、達成、打ち勝つ、ライバル、トップクラス
    2. 保障・安定
      1. 安全、安定、手堅い、着実な、パターン化、忠誠心、終身
    3. 起業家的創造性
      1. 起業、創業、事業を起こす、創意工夫、ビジネスチャンス、オーナー

5. リアリスティック・ジョブ・プレビューの意義と限界について説明する

  1. RJPとは
    1. 採用活動のときに本当の会社の姿を伝える
  2. メリット
    1. 市場型の短期適応に適した即戦力となる人材は育成できる

    2. 本当の情報がわからない
    3. 多くの人から選抜できない
    4. 優秀な人が来ない
  4. 効力を発揮する条件
    1. 建設業で効果がある傾向が認められる
    2. 社内公募
    3. 管理職登用
    4. 中途採用
    5. 人材派遣

6. 組織文化とは何か、エドガー・シャインの考え方やモデルを図示しながら具体例と共に説明する

  1. 組織文化とは何か
  2. 組織に共有された、規範、価値観、信念
  3. 会社によって「これが正しい」と考えられているルール
  4. 組織文化の3つのレベル
    1. 人工物
      1. 創りだされた物理的・社会的環境
      2. 建物、オフィスのレイアウト、服装、社員の振る舞い、理念、行事
    2. 価値
      1. 議論されたり疑問視されたり反対されることもあるもの
      2. リーダーが何か主張したとすれば、それはリーダーの信念に基づいた価値の表明
      3. 構成員には納得している人としていない人がいる
    3. 基本的仮定
      1. 対立もなく議論の余地もない無意識なもの
      2. 議論され得る「価値」が、成功体験によって、徐々に「仮定」になる
  5. 基本的仮定の5項目
    1. 現実と真実の本質：時間
    2. 現実と真実の本質：空間
    3. 人間性の本質
    4. 人間活動の本質
    5. 人間関係の本質

7. リーダーシップを定義し、望ましいリーダーシップ行動を図示しながらフォロワーの成熟度別に論じる

  1. リーダーシップとは
    1. ついてくる人がいる
    2. ついてゆく人が自律的である
  2. （Z順に）支援的行動（Y）×指示的行動（X)別の効果的リーダー行動とフォロワーのレディネス
    1. 支援型（低指示／高支援）
      1. 中Bレディネス：能力は高いが、意欲が弱く不安を残す
      2. ◯：参画・励まし・支持支援・権限付与
      3. ✕：保護・懐柔・へりくだり・なだめる
    2. コーチ型（高指示／高支援）
      1. 中Aレディネス：能力は低いが、意欲や確信を示す
      2. ◯：売り込み・説明・明確化・説得
      3. ✕：操作・説教・言い訳・合理化
    3. 委任型（低指示／低支援）
      1. 高レディネス：能力が高く、意欲や確信を示す
      2. ◯：委任・観察・信頼・依頼
      3. ✕：放任・放てき・回避・引っ込み
    4. 指示型（高指示／低支援）
      1. 低レディネス：能力も意欲も低く、不安を示す
      2. ◯：教示・ガイド・指図・決めてしまう
      3. ✕：要求・けなす・支配・責める

9. グループシンクが起きる原因とその解消方法を説明する

  1. グループシンクの定義
    1. 凝集性の高い集団において発生する
    2. 集団内の合意を得ようと意識するあまり、意思決定が非合理的な方向に歪められる
  2. グループシンクの兆候
    1. 対立相手を潰すことに力を入れる
    2. 議論の手間を惜しもうとする
  3. グループシンクの原因
    1. リーダーの影響力が強い
    2. 外部から孤立している
    3. 時間的プレッシャーがある
    4. 秩序立った意思決定手続きがない
  4. グループシンクの解決法
    1. 偏りのないリーダーの任命・育成
    2. 意見に対し多くの批評ができるよう議論をリードする
    3. 集団としての目的を意識させ他者の意見に対して偏見を持たずに評価する
    4. 議論のプロセスの工夫

12. 職能別、事業部別、マトリックス組織とは何か、図示しながら功罪を比較し説明する

  1. 職能別組織とは
    1. 生産、販売、研究開発、購買、財務、経理、人事など
    2. 同種の専門職能ごとに部門化を行った組織
    3. 長所：専門家の利益、規模の経済
    4. 短所：評価の難しさ（評価基準の違い）、トップの負担（部門間調整）、部門間対立
  2. 事業部別組織とは
    1. 事業別、製品別、あるいは地域別に複数の自律的単位に分割されている
    2. 事業部とは独立した本社機構が設けられている
    3. 長所：部門評価の明確さ、健全な内部競争、トップの長期的視座
    4. 短所：設備・投資・人員・知識などの無駄、短期的な業績優先、部門間に落ちる・またがる仕事
  3. マトリックス組織とは
    1. 2ボスシステム
    2. バランスを取る方法
      1. 職能部門長と事業部門長が腹を割って話し合う
      2. 両者のコンフリクトをトップが強権によって解消
      3. ミドル自身が葛藤をしながら適宜バランスをとる
      4. コンフリクトに強いタフなミドルが必要

14. ビジョナリーカンパニーとはどのようなものか、その特徴と経営者の役割、具体例を説明する

  1. ビジョナリー・カンパニーとは
    1. 長期に渡って卓越した企業
    2. しっかりとした基本的理念（基本的価値観＋目的）がある
    3. 基本的価値観：組織にとって不可欠で普遍の主義
    4. 目的：単なるカネ儲けを超えた会社の根源的な存在理由
  2. 経営者の役割とは
    1. 時を告げる
      1. みんな俺についてこいタイプ
      2. 社長から答えを教える
    2. 時計を作る
      1. きみはどう思うタイプ
      2. 社員に自ら答えを見つけさせる
